A cinematic, scrollable story page built for the UTS Tech Festival booth.
It presents the VisionScape project — an AI system that converts indoor 3D scene data
into structured 2D floorplans using deep learning.
Designed to run on a laptop display and be readable from a distance.

## What the Website Contains

- **Challenge** — Why floorplan creation is slow, expensive, and hard to scale
- **Our Approach** — The VisionScape pipeline from scene ingestion to SVG output
- **Core Technology** — Inside RoomToFP v7: CNN encoder, Transformer decoder, Mask decoder
- **Research Journey** — Seven architecture versions (V1 → V7), what worked and what didn't
- **Detection Capabilities** — Rooms, walls, openings, labels, polygons, and export formats
- **Honest Science** — Current limitations and future research directions
